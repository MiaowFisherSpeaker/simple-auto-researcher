# 用户偏好
## 联网搜索偏好
* 联网搜索优先级：
    1. acpx方式
       1. 同一个session 使用持久会话，每次自动 /clear.操作步骤如下
          1. 如果没有session: `acpx opencode sessions new --name search`
          2. 连接session: `acpx opencode session [session_id]`
          3. 清空上下文: `/clear`
          4. 执行搜索: "your prompt"
       2. Fallback：`acpx opencode exec` - 一般第一种行不通时使用。使用一次性对话。缺点，浪费context，以及缓存命中率低。
    2. Tavily skill - 其次（isolated沙箱环境禁止使用）
    3. BraveAPI - 禁止使用（未配置，isolated沙箱环境禁止使用）
    4. Chrome浏览器 - 默认禁止，需要明确指示；如果使用，默认【有头】模式
* 学术论文检索方式：
    1. Google Scholar - 优先
    2. arXiv - 其次
    3. 在会议官网检索 - 一般情况是从Google Scholar检索到论文后，才会去会议官网检索。
    4. 其他学术数据库（如：IEEE Xplore、ACM Digital Library） - 仅在Google Scholar和arXiv未找到相关论文时使用。
    5. 当且仅当阅读论文时才可以使用的工具，搜索社交媒体如知乎csdn等，获取论文的解读和讨论。

## 论文要求
* 论文时间年限：
  1. 近几年（2024-2026），如果近几年相关领域内没有高质量论文，可以适当放宽到近【5】年（2022-2026）。
  2. 如果某个领域内的奠基性工作较早（如：2018年之前），可以适当放宽到近【10】年（2014-2026），但需要在日志中注明理由。
  3. 如果近几年论文几乎阅读完毕或没有更多的产出，可以将时间继续放宽，但需要在日志中注明理由。
* 论文笔记要求：
  1. 每篇论文至少包含以下内容：研究问题、方法概述、实验结果、优缺点分析、与其他相关工作的比较。
  2. 论文笔记需要清晰明了，突出核心贡献和创新点，避免过于冗长或过于简略。
  3. 对于重要的论文，需要进行更深入的分析，包括方法细节、实验设置、结果解读等。
  4. 论文笔记需要按照一定的格式进行记录，便于后续查阅和总结。可以参考以下模板：
   - 标题、作者、年份、来源
   - 研究动机与背景
   - 方法论与核心框架
   - 主要公式
   - 数据集与评价指标
   - 实验结果（主实验 + 消融实验）
   - 附录补充内容
   - 个人疑问点（标 ❓）或者评价（标 💡）
* 论文阅读工具
  1. 模型自行决定
  2. 如果用户偏好已经设置则优先考虑用户偏好。
